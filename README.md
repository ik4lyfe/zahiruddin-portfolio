# Zahiruddin Zaki — Personal Portfolio

Personal portfolio website built with React, TypeScript, and Tailwind CSS.

🌐 Live site: https://www.zahiruddinzaki.tech/

## About

Portfolio showcasing my work in web development, data analysis, and 
operations management. Built as a passion project using AI-assisted 
development.

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS v3
- Framer Motion
- React Router v6

## Run Locally
```bash
npm install
npm run dev
```

## Deploy

Deployed automatically via Vercel on every push to main.
